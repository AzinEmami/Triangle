how to run  Triangle project from CMD:

1-Find Triangle-1.0.jar from ..\Triangle\target\Triangle-1.0
2-Copy Triangle-1.0 file into your desired path.
3-Open cmd.
4-Go to the directory which you have copied Triangle-1.0 file.  ( cd directory)
5-Run : java -jar Triangle-1.0.java 
6-And that's it :)
